# Internet Source Registry

Internet sources describe approved public/read-only endpoints such as Gamma market APIs, CLOB market/orderbook/trade read APIs, public CSV URLs, and public JSON URLs.

Sources are disabled by default. The registry records source type, base URL, endpoint path, query parameters, allowed domain, timeout, rate limit, blockers, warnings, and last run status.

Do not register private, authenticated, paywalled, or trading endpoints unless the operator has explicit authorization and the code path is read-only.
