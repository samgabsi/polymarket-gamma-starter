# Operator Notes v4.17.0-real

Use the new five-workspace model as the normal operating flow:

- Start at **Command Center** (`/v3`).
- Use **Opportunities** for trade/review candidates and source-specific AI/arbitrage detail.
- Use **Automation / Paper Trading** for paper-only strategy cycles and simulated ledger review.
- Use **Review & Audit** for accountability and local decisions.
- Use **Settings & System** for readiness, configuration, venues, diagnostics, and advanced routes.

Detailed routes are preserved for compatibility. They are no longer intended to feel like equal top-level destinations.

Safety remains unchanged: this UX consolidation does not enable live trading, real order placement, cancellation, or autonomous live execution.
